### Write-up:
Searched for a gameboy emulator, searching the web you will find an emulator called [someboy](https://sameboy.github.io/). Go to the download page, execute the file and load the .gb file inside the emulator. You can move inside the game with the arrow buttons and use 'x' and z'' as 'A' and 'B' accordingly. Then when you have created a new game and loaded in, you will find a house inside the map, go inside the flag will be visible on the floor. Flag: `HCTF-FLAG-GAME_BOY_FUN_22!`
